# 장기: 황산벌의 기억 — Development Epics

POC 2주 범위. 에픽 순서는 의존성 기반 — E1 완료 후 E2, E2 완료 후 E3.

---

## E1. 기반 구현

**목표:** 장기판 위에서 4개 기물이 룰에 맞게 이동하는 플레이 가능한 상태

**포함:**
- SVG 장기판 렌더링 (9×10 격자, 수묵화 스타일 선)
- 4개 기물(차, 마, 포, 졸) SVG 렌더링 및 배치
- 클릭 선택 → 이동 가능 칸 흐리게 표시 → 클릭 이동
- 장기 이동 룰 적용 (차/마/포/졸 각 규칙)
- 잘못된 이동 조용히 막기

**명시적 제외:**
- 애니메이션 스펙 (ms/이징) — E3
- 사운드 — E3
- 스토리/내레이션 — E2

**완료 기준:** 4개 기물이 장기 룰에 맞게 이동, 오동작 0건 (수동 테스트 통과)

---

## E2. 스토리 시스템

**목표:** 6개 장면이 순서대로 흐르는 완주 가능한 경험

**포함:**
- 장면 전환 시스템 (내러티브 → 체험 → 내러티브 순환)
- 오프닝 장면: 황산벌 배경 내레이션 텍스트 + 배경 일러스트
- 기물별 소개 장면 (×4): 기물 등장 + 캐릭터 설명 + 이동 시연 + 직접 체험
- 계백 대사 시스템: 기물 이동 완료 시 텍스트 대사 표시
- 결말 장면: 황산벌 전투 결말 내레이션 + 일러스트
- AI 생성 스토리 텍스트 + GD 편집 (역사적 정확성 검토)
- AI 생성 일러스트 에셋 + GD 큐레이션

**명시적 제외:**
- 성우 녹음 (텍스트 대사로 대체)
- 애니메이션/사운드 — E3

**완료 기준:** 오프닝부터 결말까지 6장면 완주 가능, 이탈 없이 흐름 자연스러움

---

## E3. 감각 완성

**목표:** 기둥 2(여백미)·3(감정 담은 움직임)이 체감되는 최종 POC

**포함:**
- 기물 이동 애니메이션 스펙 적용:
  - 차: 500ms, ease-in
  - 마: 300ms, ease-out
  - 포: 150ms, linear
  - 졸: 400ms, ease-in
- 사운드 4종 적용: 목재 미끄러짐 / 가벼운 타격 / 날카로운 타격 / 둔탁한 타격
- 국악기 앰비언트 BGM (가야금/해금/대금)
- 이동 가능 칸 힌트 시각 정제 (흐림 강도, 색상 조율)
- 전체 비주얼 에셋 최종 적용 (일러스트, UI 여백 조율)
- 컬러 팔레트 적용: 먹색, 한지색, 포인트 단청색

**명시적 제외:**
- 성능 최적화 심화 (60fps 달성이 목표, 심화 최적화는 v1.0)

**완료 기준:** 60fps 유지, 기물마다 감정이 다르게 느껴짐, 정성 테스트 5인 이상 완주 + 피드백 수집
