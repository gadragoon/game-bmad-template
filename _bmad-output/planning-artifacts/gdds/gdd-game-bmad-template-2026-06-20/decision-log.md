- [x] 기둥 1 확정: "플레이어는 이야기 속 장면을 통해서 장기라는 게임을 자연스럽게 문화로써 체험한다"
- [x] 기둥 2 확정: "이 게임의 모든 시각적 요소는 여백미로 말한다."
- [x] 기둥 3 확정: "기물의 움직임은 그 기물을 바라보는 시선의 감정을 담는다." (계백=절실함/처절함, 관창=도전/패기. 기물 종류가 아닌 시선이 감정을 결정)
- [x] 기둥 수 확정: 3개. 기둥 4(점진적 이해)는 기둥 1에 통합. 장기의 전략성은 기둥이 아닌 전제(바닥).
- [x] 핵심 게임 루프 확정 (POC 범위):
    역사 내레이션 → 기물 등장/소개 → 이동 시연 → 직접 움직여보기 → 계백 목소리 대사(보상) → 다음 장면
- [x] 잘못된 이동 처리: 이동 불가(조용히 막힘) + 이동 가능한 칸 흐리게 표시. 오류 피드백 없음.
- [x] POC 완료 상태: 마지막 기물 체험 후 황산벌 전투 결말 장면으로 마무리. 승/패 없음 — 완주 or 이탈만 존재.
- [x] 기물 메카닉 수치화 완료:
    차(500ms/ease-in/미끄러짐), 마(300ms/ease-out/가벼운타격), 포(150ms/linear/날카로운타격), 졸(400ms/ease-in/둔탁한목재)
- [x] 기술 스택 확정: SVG. 이유: 벡터 기반으로 여백미 표현 적합, CSS/GSAP 애니메이션으로 기물 이동 스펙 구현, 2주 POC 현실성 높음.
- [x] POC 기물 확정: 차(車), 마(馬), 포(砲), 졸(卒)
- [x] 장면 구조 확정: 6장면 (오프닝 + 기물 체험 4개 + 결말). 장면 1:체험 1 균형 비율.
